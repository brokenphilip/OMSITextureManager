# OMSITextureManager
An OMSI 2 Bus Simulator plugin/tool which gives insight on, and aims to optimize, texture memory usage.

> [!WARNING]
> This project is superseded by [BulbToys/OMSI](https://github.com/BulbToys/OMSI)'s Texture Manager module, which integrates and improves upon OMSITextureManager's functionality.
